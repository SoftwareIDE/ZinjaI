En español:
===========
Los temas de iconos son simplemente conjuntos de imagenes. Para crear un nuevo tema, simlemete debe crear un subdirectorio aqui, y un archivo descripcion.txt dentro del mismo. El archivo txt debe contener el texto que se mostrará como descripción en la ventana de selección de tema (usualmente los créditos). 
En el directorio, puede colocar un reemplazo para cualquiera de las imagenes que ve en el directorio imgs. 
En la mayoría de los casos se debe respetar el tamaño (aún cuando no hacerlo parezca no generar problemas, algunos errores no se presentan en todos los sistemas operativos). Es importante además respetar el uso de mayúsculas y minúsculas en los nombres de archivos, ya que algunos sistemas operativos distiguen mayusculas y minusculas. 
Para las barras de herramientas, los iconos deben estar en subdirectorios según el tamaño (16,32,24,48 o 64 son los tamaños válidos, no necesariamente deben estar todos). Las imágenes del subdirectorio 16 también se utilizan en los menúes.
Para las imágenes que no se incluyan, en algunos casos se utilizará la del tema original, en otros no se utilizará ninguna.
También se puede reemplazar el panel de bienvenida con un conjunto de archivos html de nombre welcome_panel_(nombre de idioma).
Si usted crea un nuevo tema y desea verlo incluido en la versión oficial de ZinjaI, envíelo a zaskar_84@yahoo.com.ar. Si tiene dudas o sugerencias sobre algún aspecto de la gestión de temas, también puede escibir a la misma dirección.
En el directorio del tema por defecto (imgs) encontrará el archivo svg (puede abrirlo con un editor vectorial, por ejemplo Inkscape) con el cual se generaron todas las imágenes de dicho tema (en windows, debe instalar los fuentes de ZinjaI).


In English:
===========
Skins are in fact only image packages. If you want to make a new skin, just create a folder here, and a file descripcion.txt. The txt file should contain any text you want to display in skin selection dialog (usally credits). 
In the folder you can place any image file that you see in imgs folder to replace it. 
You must keep the same image size for most of them (even if changing it doesn't seem to generate troubles, errors are some time platform-dependent). It is also important to preserve case in file names (some operative sistems do have case-sensitive file systems). 
For toolbar images, you can make different sizes (16,24,32,48 and 64 are valid ones, and it is not necessary to have all of them). Icons fo size 16 will also be used for menues.
For the images you don't replace, sometimes the original will be used, sometimes no image will. 
You can place also a different welcome panel by writing a new welcome_panel_(language name).html file. 
If you make a new skin and you'd like to see it in the next ZinjaI's release, send it to zaskar_84@yahoo.com.ar. If you have doubts or suggestions, you can also write to the same mail address.
In the default skin folder (imgs) you can find the svg file (to opened with a vector drawing program, for instance: Inkscape) wich was used to generate all the images for that skin (in Widows version, you must install sources to get it).
