Estructura tm

Contiene una fecha y hora determinada separada en sus componentes.

Miembros:
  • int tm_sec
  • int tm_min
  • int tm_hour
  • int tm_mday
  • int tm_mon
  • int tm_year
  • int tm_wday
  • int tm_yday
  • int tm_isdst


Se encuentra en: <ctime>

    Este tipo de dato está pensado para representar fechas y horas determinadas separando los componentes de las mismas. La hora se divide en los miembros tm_hour, tm_min y tm_sec que representan horas, minutos y segundos respectivamente (de 0 a 23 para la hora, y de 0 a 59 para minutos y segundos). La fecha se divide en tm_mday, tm_mon y tm_year que representan el dia del mes, la cantidad de meses transcurridos desde enero y la cantidad de años transcurridos desde 1900 respectivamente (de 1 a 31 para el dia y de 0 a 11 para el mes). Además, e tm_wday indica el dia de la semana (de 0 a 6 correspondiendose con domingo a sabado), tm_yday indica el dia dentro del año (de 0 a 365) y tm_isdst indica si la hora de verano esta activa o no (es positivo cuando esta activa, cero cuando no, y negativo si se desconoce).


Ejemplo:
    time_t t1 = time(NULL);
    tm *ptm2 = localtime( &t1 );
    char * dias[] = { "Domingo", "Lunes", "Martes", "Miercoles", "Jueves", "Viernes", "Sabado" };
    cout << "Hoy es " << dias[ptm2->tm_wday] << endl;

  • Función time: Devuelve la fecha/hora actual en segundos.
  • Tipo time_t: Representa una fecha y hora determinada en segundos.
  • Tipo clock_t: Representa una cantidad de ticks del reloj.
  • Función mktime: Convierte una fecha y hora de una estructura tm al tipo time_t.
