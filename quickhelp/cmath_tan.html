Función tan

Calcula la tangente de x.

Prototipos:
  • double tan ( double x );
  • float tan ( float x );
  • long double tan ( long double x );


Se encuentra en: <cmath>

    Devuelve la tangente de un arco de x radianes.


Ejemplo:
    float x = tan(M_PI*1.5);
    float y = atan(x);
