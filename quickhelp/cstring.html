Cabecera <cstring>

Contiene funciones para el manejo de cadenas de caracteres c y arreglos.

Funciones:
  • Función strlen: Devuelve la longitud de una cadena.
  • Función strcmp: Compara cadenas.
  • Función strncmp: Compara partes de cadenas.
  • Función strcpy: Copia una cadena en otra.
  • Función strncpy: Copia parte de una cadena en otra.
  • Función strcat: Concatena cadenas.
  • Función strncat: Concatena una cadena y parte de otra.
  • Función strchr: Busca la primer ocurrencia de un caracter dentro de una cadena.
  • Función strrchr: Busca la última ocurrencia de un caracter dentro de una cadena.
  • Función strstr: Busca una cadena dentro otra.
  • Función strspn: Busca en una cadena el primer caracater no perteneciente a un conjunto dado.
  • Función strcspn: Busca en una cadena el primer caracater perteneciente a un conjunto dado.
  • Función strtok: Separa una cadena en partes según un conjunto de caracateres delimitadores.
