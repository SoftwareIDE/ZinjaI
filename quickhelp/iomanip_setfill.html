Manipulador setfill

Define el caracter de relleno.

Prototipos:
  • smanip setfill ( char c );


Se encuentra en: <iomanip>

    Define el caracter de relleno (c) utilizado para completar la longitud de un dato a insertar en el flujo cuando esta es menor que la definida por el ancho de campo (equivale a llamar al método fill de la clase ios). Para definir el ancho de campo se puede utilizar el manipulador setw, y para definir en que posición de la cadena a insertar se colocan los caracteres de relleno se pueden utilizar los manipuladores rigth, internal y left.


Ejemplo:
   int a[]={-100, -10,-1,10,100};
   
   cout<<" left | internal | right "<<endl;
   cout<<setfill('-')<<setw(26)<<""<<endl;
   cout<<setfill(' ');
   
   for (int i=0; i<5;i++) {
      cout<<" "<<setw(5)<<left<<a[i]<<" | ";
      coutsetw(6)<<internal<<a[i]<<" | ";
      cout<<setw(5)<<right<<a[i]<<endl;
   }
