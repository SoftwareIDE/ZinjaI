Función calloc

Reserva e inicializa dinámicamente memoria para un arreglo.

Prototipo:
  • void *calloc( size_t cant, size_t tam );


Se encuentra en: <cstdlib>

    Reserva un bloque de memoria cuyo tamaño en bytes está dado por el producto entre cant y tam, por lo que se utiliza normalmente para crear un arreglo de cant elementos donde cada elemento ocupa tam bytes). Además inicializa cada byte de dicha memoria con 0. Devuelve un puntero de tipo void* al comienzo del bloque, o NULL cuando no ha podido reservarlo. Normalmente el puntero deberá ser convertido (casting) al tipo de dato requerido. Esta memoria deberá ser liberada con free cuando ya no se utilice.

Ejemplo:
    int *arreglo = (int*) calloc(100,sizeof(int));
    /* ... */
    free (arreglo);
