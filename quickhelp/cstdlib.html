Cabecera <cstdlib>

Contiene funciones generales, incluyendo funciones para el manejo de memoria, conversion de tipos, control de proceso, etc.

Funciones:
  • Función atoi: convierte una cadena (char *) en un entero tipo int.
  • Función atof: convierte una cadena (char *) en un numero tipo double.
  • Función atol: convierte una cadena (char *) en un entero tipo long int.
  • Función getenv: devuelve el valor de una variable de entorno.
  • Función system: Ejecuta un comando o programa externo.
  • Función malloc: Reserva dinámicamente un bloque de memoria.
  • Función calloc: Reserva e inicializa dinámicamente memoria para un arreglo.
  • Función realloc: Cambia el tamaño de un bloque de memoria reservado dinámicamente.
  • Función free: Libera un bloque de memoria reservado dinámicamente.
  • Función rand: genera un número pseudo-aleatorio.
  • Función srand: Inicializa la semilla del generador de números pseudo-aleatorios.
  • Función exit: Finaliza el programa.
