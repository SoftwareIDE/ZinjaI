Este error suele ocurrir porque el sistema de archivos en donde se está intentando colocar el ejecutable no tiene permisos de ejecución. En GNU/Linux, un archivo es ejecutable si tiene su permiso de ejecución seteado. Este permiso es un atributo, similar al atributo de oculto, o de sistema en otros sistemas de archivos como los que utiliza Windows (fat32, ntfs). Si el sistema de archivos no contempla este atributo/permiso, Linux asigna a todos los archivos de dicho sistema el mismo valor (todos son ejecutables, o ninguno es ejecutable), y este valor depende de las opciones de montaje. Puede modificar las opciones de montaje de su sistema para permitir la ejecución, o mover su proyecto/fuente a una carpeta de una unidad con otro sistema de archivos.
