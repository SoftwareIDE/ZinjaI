Función malloc

Reserva dinámicamente un bloque de memoria.

Prototipo:
  • void *malloc( size_t tam );


Se encuentra en: <cstdlib>

    Reserva un bloque de memoria cuyo tamaño en bytes está dado por tam. Devuelve un puntero de tipo void* al comienzo de dicho bloque, o NULL cuando no ha podido reservarlo. Normalmente el puntero deberá ser convertido (casting) al tipo de dato requerido. Esta memoria deberá ser liberada con free cuando ya no se utilice.

Ejemplo:
    int *arreglo = (int*) malloc(100*sizeof(int));
    /* ... */
    free (arreglo);


Vea tambien:
  • Cabecera cstdlib: Contiene funciones generales, incluyendo funciones para el manejo de memoria, conversion de tipos, control de proceso, etc.
  • Función calloc: Reserva e inicializa dinámicamente memoria para un arreglo.
  • Función realloc: Cambia el tamaño de un bloque de memoria reservado dinámicamente.
  • Función free: Libera un bloque de memoria reservado dinámicamente.
