Función clock

Devuelve la cantidad de ticks de reloj transcurridos desde el inicio del programa.

Prototipo:
  • clock_t clock ( void );


Se encuentra en: <ctime>

    Devuelve la cantidad de ticks de reloj transcurridos desde el inicio del programa. Para convertir esta cuenta a segundos se utiliza la macro CLOCKS_PER_SEC, ya que la cantidad de ticks por segundos varía según arquitectura y sistema operativo. En algunos sistema esta cuenta puede no incrementarse durante los tiempos en que el proceso permanece bloqueado (por ejemplo, a la espera de una entrada de usuario o de la liberación de un recurso).


Ejemplo:
    long int nro;
    cout<<"Ingrese un nro:";
    cin>>nro;
    clock_t t1 = clock();
    for (long int i=0;i<10000;i++)
        for (long int j=0;j<nro;j++)
            ;
    clock_t t2 = clock();
    cout << "Contar diez mil veces hasta " << nro << " toma " << float(t2-t1)/CLOCKS_PER_SEC << " segundos.";
