Función frexp

Descompone un número de punto flotante en mantisa y exponente.

Prototipos:
  • double frexp ( double x, int *e );
  • float frexp ( float x, int *e );
  • long double frexp ( long double x, int *e );


Se encuentra en: <cmath>

    Devuelve los valores correspondientes a la mantisa y el exponente que representan el numero de punto flotante x (x=mantisa*2^exponente). La mantisa es el valor de retorno de la función, mientras que el exponente se devuelve a travez del parámetro e. Si x es 0, tanto mantisa como exponente valen 0.
    Esta forma de representar un número está directamente ligada a la forma en que se almacenan en memoria los número de punto flotante (como float o double).


Ejemplo:
    float num=3.5, mantisa;
    int exponente;
    mantisa = frexp(num,&exponente);
    float num_otra_vez = ldexp(mantisa,exponente);
