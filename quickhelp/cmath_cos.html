Función cos

Calcula el coseno de x.

Prototipos:
  • double cos ( double x );
  • float cos ( float x );
  • long double cos ( long double x );


Se encuentra en: <cmath>

    Devuelve el coseno de un arco de x radianes.


Ejemplo:
    float x = cos(M_PI*1.5);
    float y = acos(x);
