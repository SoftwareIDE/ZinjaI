Función floor

Redondea hacia abajo.

Prototipos:
  • double floor ( double x );
  • float floor ( float x );
  • long double floor ( long double x );


Se encuentra en: <cmath>

    Devuelve el valor de x redondeado hacia abajo. Es decir, que devuelve el entero mas cercano menor o igual a x.

Ejemplo:
    float cinco = floor(5.9);
    float menos_seis = floor(-5.9);
