namespace

    Los espacios de nombres en C++ son un concepto que permite agrupar un conjunto de declaraciones bajo un ámbito común e independiente. Utilizando la sintaxis namespace nombre { ... } se puede declarar clases, variables o funciones como pertenecientes a este scope(denominado por nombre). Ej:

    namespace mi_espacio {
       int funcion1();
       int funcion2();
       float var1;
       float var2;
       class clase1;
       ...
    }

    Para utilizar los elementos de este espacio de nombres se debe utlizar el operador de resolución de scope (::) o la directiva using. Ej:

    int a = mi_espacio::funcion1();


    Dado que todas las clases y funciones del lenguaje ANSI/ISO C++ actual están definidas dentro de un espacio de nombres denominado std, es muy común que la primer linea a continuación de la lista de #includes sea using namespace std;.
