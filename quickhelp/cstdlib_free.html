Función free

Libera un bloque de memoria reservado dinámicamente.

Prototipo:
  • void srand ( unsigned int seed );


Se encuentra en: <cstdlib>

    Libera un bloque de memoria previamente reservado con calloc, malloc o realloc. Recive un puntero al comienzo del bloque de memoria. No debe usarse para liberar memoria reservada por otros métodos (por ejemplo con el operador new).

Ejemplo:
    int *arreglo = (int*) calloc(100,sizeof(int));
    /* ... */
    free (arreglo);


Vea tambien:
  • Cabecera cstdlib: Contiene funciones generales, incluyendo funciones para el manejo de memoria, conversion de tipos, control de proceso, etc.
  • Función malloc: Reserva dinámicamente un bloque de memoria.
  • Función calloc: Reserva e inicializa dinámicamente memoria para un arreglo.
  • Función realloc: Cambia el tamaño de un bloque de memoria reservado dinámicamente.
