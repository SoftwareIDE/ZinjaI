return

    La palabra clave return se utiliza para indicar el valor de retorno dentro de una función o método y detener su ejecución. Es decir, que al llegar el control del programa a esta linea, devolverá el valor que se indique a continuación del return y finalizará la ejecución de dicha función o método. El valor indicado puede ser una variable, o una expresión. En cualquier caso debe coincidir en tipo con el tipo de retorno declarado para la función, o poder ser convertido mediante un constructor de dicho tipo (por ejemplo, return 5; en una función declarada como float convertirá automáticamente el entero 5 al flotante equivalente mediante el constructor float(int)). Dentro de una función pueden existir más de un punto de retorno. Si la función es declarada void se utiliza return; (sin ningun valor) o bien puede omitirse; en caso contrario su presencia es obligatoria. Ej:

    int factorial (unsigned int a) {
        if (a==0)
            return 1;
        else
            return factorial(a-1)*a;
    }
    ...
    flaot r = factorial(5);
