Objeto cout

Representa el flujo de salida estándar.

Declaración:
  • extern ostream cout;


Se encuentra en: <iostream>

    Este objeto representa la salida estándar. Controla un flujo asociado al objeto stdout declarado en <cstdio>. Normalmente esta salida está directamente asociada a la consola, aunque el sistema operativo podría redireccionarla (por ejemplo, invocando al programa como "programa > archivo" desde el shell).
    Como todo en objeto ostream, permite escribir información utilizando el operador de redirección de flujo (<<). Se pueden enviar objetos de tipos base directamente, o instancias de clases definidas por el usuario mediante la sobrecarga de este operador. Se pueden utilizar además manipuladores de flujo para dar formato a la salida.
    Cabe aclarar que el objeto cout utiliza un buffer intermedio, y la sincronización entre el buffer del objeto y la consola puede no ser inmediata, por lo cual para mostrar información de log y/o depuración un programador preferirá utilizar los objetos clog y cerr (este último no utiliza buffer). Para forzar la sincronización de la salida estándar se pueden utilizar los manipuladores endl o flush.


Ejemplo:
    float dato[5]={9.32,12.23,6,1.3,7.4535};
    char *nombre[5]={"Horacio","Horge","Gerardo","Pablo","Federico"};
    cout<<"Hola Mundo, aqui va un tabla:"<<endl;
    for (int i=0;i<5;i++)
        cout<<setfill('.')<<setw(15)<<nombre[i]<<fixed<<setprecision(2)<<dato[i]<<endl;

  • Cabecera iostream: Contiene clases y objetos para el manejo de flujos de entrada y salida.
  • Cabecera iomanip: Contiene manipuladores de flujo para dar formato a flujos de salida.
  • Clase ostream: Modela un flujo de salida genérico.
  • Objeto cerr: Representa el flujo de salida de error.
  • Objeto clog: Representa el flujo de salida de log.
  • Objeto cin: Representa el flujo de entrada estándar.
