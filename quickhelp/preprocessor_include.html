Directiva de preprocesador: #include

Incluye el código de otro archivo.

Utilización:
  • #include "nombre_archivo.h"
  • #include <nombre_archivo.h>


    Indica al preprocesador del compilador que debe reemplazar esa linea por el contenido del archivo indicado por nombre_archivo. Si el nombre del archivo se encuentra entre comillas, el compilador buscará dicho archivo en la mimsa carpeta que el fuente que contiene la directiva #include, y si no lo encuentra lo buscará luego entre las cabeceras estándar del sistema. Si el nombre del archivo está entre signos menor y mayor, el compilador buscará el archivo directamente entre las cabeceras estándar del sistema.

Ejemplo:
    #include "miClase.h"
    #include <iostream>
