Manipulador setw

Define el ancho de campo para la siguiente inserción.

Prototipos:
  • smanip setw ( int n );


Se encuentra en: <iomanip>

    Define el ancho de campo para la siguiente inserción (equivale a llamar al metodo width de la clase ios_base con n como argumento). El ancho de campo representa la minima cantidad de caracters que se escribiran en la proxima inserción. Es decir, que cuando se inserte contenido al flujo, si la longitud del contenido es menor a n se rellenará hasta que su longitud sea efectivamente igual a n. El caracter con que se rellena se puede definir con el manipulador setfill y la posición en la que se inserta dicho caracter puede definirse con los manipuladores rigth, internal y left.


Ejemplo:
   int a[]={-100, -10,-1,10,100};
   
   cout<<" left | internal | right "<<endl;
   cout<<setfill('-')<<setw(26)<<""<<endl;
   cout<<setfill(' ');
   
   for (int i=0; i<5;i++) {
      cout<<" "<<setw(5)<<left<<a[i]<<" | ";
      cout<<setw(6)<<internal<<a[i]<<" | ";
      cout<<setw(5)<<right<<a[i]<<endl;
   }
