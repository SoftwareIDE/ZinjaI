Función getenv

Devuelve el valor de una variable de entorno.

Prototipo:
  • char *getenv ( const char* nomvar );


Se encuentra en: <cstdlib>

    Recibe una cadena de caracteres c conteniendo el nombre de una variable de entorno del sistema, y devuelve un puntero a su contenido (tambien una cadena de caracteres c) si existe, o NULL en caso contrario. Este puntero no sirve para modificar el contenido de la variable en cuestión.

Ejemplo:
    char *valor;
    valor = getenv("PATH");
    if (valor)
       cout<<"PATH = "<<valor<<endl;
    else
       cout<<"La variable PATH no esta definida"<<endl;
