Función fmod

Calcula el resto de una división.

Prototipos:
  • double fmod ( double x );
  • float fmod ( float x );
  • long double fmod ( long double x );


Se encuentra en: <cmath>

    Devuelve el resto de la división x/y. Es equivalente a utilizar el operdador % (x%y).

Ejemplo:
    float uno = fmod(10.f,3);
