Función sqrt

Calcula una raiz cuadrada.

Prototipos:
  • double sqrt ( double x );
  • float sqrt ( float x );
  • long double sqrt ( long double x );


Se encuentra en: <cmath>

    Calcula la raiz cuadrada de x. x debe ser positivo. Si x es negativo el resultado es el valor especial IND.
    En algunas circunstancias el compilador puede no llegar a determinar cual de las sobrecargas de esta función utilizar. En caso de ambigüedad, existen las funciones sqrtf y sqrtl para forzar los datos a tipos double y long double respectivamente, aunque estas dos funciones no son estandar.
    Para calcular otras raices utilice la funcion pow (considerando que la raiz i-esima de b es b^(1/i) ).

Ejemplo:
    float raiz_de_2 = sqrt(2);
    float cinco = sqrt(25);
