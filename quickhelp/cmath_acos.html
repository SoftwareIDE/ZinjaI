Función acos

Calcula el arco coseno de x.

Prototipos:
  • double acos ( double x );
  • float acos ( float x );
  • long double acos ( long double x );


Se encuentra en: <cmath>

    Devuelve el arco coseno x, expresado radianes. Si el resultado es demasiado grande para el tipo de dato devuelto la función devuelve HUGE_VAL.


Ejemplo:
    float x = cos(M_PI*1.5);
    float y = acos(x);
