Función abs

Devuelve el valor absoluto de x.

Prototipos:
  • double abs ( double x );
  • float abs ( float x );
  • long double abs ( long double x );


Se encuentra en: <cmath>

    Devuelve el valor absoluto de x (esto es , x si x era 0 o positivo, -x en caso contrario).


Ejemplo:
    float cinco = abs(-5);
