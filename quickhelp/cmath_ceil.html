Función ceil

Redondea hacia arriba.

Prototipos:
  • double ceil ( double x );
  • float ceil ( float x );
  • long double ceil ( long double x );


Se encuentra en: <cmath>

    Devuelve el valor de x redondeado hacia arriba. Es decir, que devuelve el entero mas cercano mayor o igual a x.

Ejemplo:
    ceil seis = floor(5.1);
    ceil menos_cinco = floor(-5.1);
