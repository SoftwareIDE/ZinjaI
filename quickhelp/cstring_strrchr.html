Función strrchr

Busca la última ocurrencia de un caracter dentro de una cadena.

Prototipos:
  • const char * strchr ( const char * str, int chr );
  • char * strchr ( char * str, int chr );


Se encuentra en: <cstring>

    Busca el caracter chr dentro de str comenzando desde el final de la cadena (desde el caracter de terminación '\0') y devuelve un puntero a la última ocurrencia si lo encuentra, o NULL en caso contrario.


Ejemplo:
    char cadena[256]="Hola Mundo!\n";
    char *res=strrchr(cadena,'o');
    bool hay_una_o=(res!=NULL);
    if (hay_una_o)
        int pos_ultima_o=res-cadena+1;


Vea tambien:
  • Cabecera cstring: Contiene funciones para el manejo de cadenas de caracteres c y arreglos.
  • Función strchr: Busca la primer ocurrencia de un caracter dentro una cadena.
  • Función strstr: Busca una cadena dentro de otra.
