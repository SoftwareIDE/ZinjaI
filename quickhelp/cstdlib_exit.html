Función exit

Finaliza el programa.

Prototipo:
  • void exit ( int estado );


Se encuentra en: <cstdlib>

    Finaliza el programa retornando al sistema el código de salida determinado por estado. Usualmente un código igual a cero indica que el programa se ejecutó correctamente, mientras que un código distinto de cero puede indicar un error. Los posibles valores válidos dependen del sistema operativo.

Ejemplo:
    if (error)
       exit(1);
    else
       exit(0);
