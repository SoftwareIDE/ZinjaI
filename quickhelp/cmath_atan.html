Función atan

Calcula el arco tangente de x.

Prototipos:
  • double atan ( double x );
  • long double atan ( long double x );
  • float atan ( float x );


Se encuentra en: <cmath>

    Devuelve el arco tangente (operación trigonométrica inversa a la tangente) de x expresado en radianes. El resultado se encuentra entre [-pi/2;+pi/2], ya que no es posible determinar el cuadrante a partir de x (para determinar el cuadrante utilice atan2).


Ejemplo:
    float x = tan(M_PI*1.5);
    float y = atan(x);
