Función strchr

Busca la primer ocurrencia de un caracter dentro de una cadena.

Prototipos:
  • const char * strchr ( const char * str, int chr );
  • char * strchr ( char * str, int chr );


Se encuentra en: <cstring>

    Busca el caracter chr dentro de str comenzando desde el primer caracter de la cadena y devuelve un puntero a la primer ocurrencia si lo encuntra, o NULL en caso contrario.


Ejemplo:
    char cadena[256]="Hola Mundo!\n";
    char *res=strchr(cadena,'o');
    bool hay_una_o=(res!=NULL);
    if (hay_una_o)
        int pos_primer_o=res-cadena+1;
