Función rand

Genera un número pseudo-aleatorio.

Prototipo:
  • int rand ( void );


Se encuentra en: <cstdlib>

    Genera un número entero pseudo-aleatorio entre 0 y RAND_MAX. Pseudo-aleatorio significa que los números no son verdaderamente aleatorios ya que la PC utiliza una función matemática para generarlos, pero las características de esta función hacen que la secuencia sea aparentemente caótica e impredecible. Para inicializar este generador de números psuedo-aleatorios se utiliza la función srand
    Un uso frecuente consiste es inicializar utilizar esta función en conjunción con el operador módulo (&mod;) para generar menores que un límite superior dado. Si lo que se desea es generar números dentro de un rango determinado ([lim_sup;lim_inf]), se suele utilizar de la siguiente forma: lim_inf+rand()%(lim_sup-lim_inf+1). Si bien estas dos últimas formas no son estadísticamente correctas, ya que si la amplitud del rango nos es divisor de RAND_MAX los números del comienzo del intervalo tendrán más probabilidad de aparecer. Sin embargo, si la amplitud del intervalo es muy pequeña en comparación con RAND_MAX esta diferencia es despreciable.


Ejemplo:
    // generar 10 numeros aleatorios entre 70 y 90
    srand(time(NULL));
    for (int i=0;i++;i<10)
       cout<<rand()%21+70<<" ";
