Función cosh

Calcula el coseno hiperbólico de x.

Prototipos:
  • double cosh ( double x );
  • float cosh ( float x );
  • long double cosh ( long double x );


Se encuentra en: <cmath>

    Devuelve el coseno hiperbólico de x.


Ejemplo:
    float y = cosh(x);
