Función gmtime

Convierte una fecha y hora time_t a una fecha y hora tm UTC.

Prototipo:
  • struct tm * gmtime ( const time_t * timer );


Se encuentra en: <ctime>

    Convierte la fecha y hora apuntada por timer (en segundos) a su equivalente para la estructura tm (dividida en componentes). La fecha y hora resultantes son fechas y horas UTC (universales), por lo que, de acuerdo a la configuración del sistema, suele diferir con la fecha y hora local. Devuelve un puntero a una estructura de tipo tm. Esta estructura esta alocada estáticamente y es compartida pos las funciones gmtime y localtime, por lo que cualquier llamada a cualquiera de ellas la sobreescribirá.


Ejemplo:
    time_t t1 = time(NULL);
    tm *ptm2 = gmtime( &t1 );
    cout << "Fecha actual (UTC): " << ptm2->tm_mday << "/" << ptm2->tm_mon+1 << "/" << ptm2->tm_year+1900 << endl;
    cout <<"Hora actual (UTC): " << ptm2->tm_hour << ":" << ptm2->tm_min << ":" << ptm2->tm_sec << endl;


Vea tambien:
  • Cabecera ctime: Contiene funciones y estructuras para el manejo de fechas y horas.
  • Función time: Devuelve la fecha/hora actual en segundos.
  • Función localtime: Convierte una fecha y hora time_t a una fecha y hora tm local.
  • Tipo time_t: Representa una fecha y hora determinada en segundos.
  • Tipo tm: Contiene una fecha y hora determinada separada en sus componentes.
