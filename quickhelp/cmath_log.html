Función log

Calcula el logaritmo natural de x.

Prototipos:
  • double log ( double x );
  • float log ( float x );
  • long double log ( long double x );


Se encuentra en: <cmath>

    Calcula el logaritmo natural de x. El logaritmo natural utiliza como base al número e. El argumento debería ser positivo.
    Matemáticamente, el logaritmo natural de x es un número y tal que e^y=x. Para encontrar un logaritmo de x en una base arbritaria b, se puede utilizar log(x)/log(b).


Ejemplo:
    float res = log(10);
