using

    La palabra clave using se utiliza para indicarle al compilador que vamos a utilizar determinados elementos. Se puede utilizar para indicar un elemento en particular, o todo un conjunto de ellos. La caracteristica de estos elementos es que se encuentran definidos dentro de un espacio de nombres, por lo que en principio no podrían ser utilizados sin incluir al llamarlos su espacio de nombres mediante el operador de resolución de scope (::). Ej:

    std::cout<<"Hola Mundo"<<std::endl;

    Al utilzar la sintaxis using name espacio::elemento se le indica al compilador que se utilizará ese elemento de dicho espacio; por lo que cuando invoquemos al elemento, si no lo encuentra en el espacio donde fue invocado, utilizará el que se encuentra en el espacio anteriormente definido. Ej:

    using std::cout;
    ...
    cout<<"Hola Mundo"<<std::endl;

    Finalmente, la forma más comun de utilizar es using namespace espacio. De esta forma le indicamos al compilador que podemos utilizar cualquier elemento dentro de ese espacio. Ej:

    using namespace std;
    ...
    cout<<"Hola Mundo"<<endl;


    Dado que todas las clases y funciones del lenguaje ANSI/ISO C++ actual están definidas dentro de un espacio de nombres denominado std, es muy común que la primer linea a continuación de la lista de #includes sea using namespace std;.
