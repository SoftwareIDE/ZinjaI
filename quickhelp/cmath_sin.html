Función sin

Calcula el seno de x.

Prototipos:
  • double sin ( double x );
  • float sin ( float x );
  • long double sin ( long double x );


Se encuentra en: <cmath>

    Devuelve el seno de un arco de x radianes.


Ejemplo:
    float x = sin(M_PI*1.5);
    float y = asin(x);
