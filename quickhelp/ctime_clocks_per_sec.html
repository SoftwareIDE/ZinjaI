Macro CLOCKS_PER_SEC

Define la cantidad de ticks de reloj por segundo.

Prototipo:
  • time_t time ( time_t *timer );


Se encuentra en: <ctime>

    Representa la cantidad de ticks de reloj por segundo correspondientes al tipo clock_t. Esta cantidad varia de acuerdo a la arquitectura y el sistem operativo utilizado. Notar que este número en realidad es solo una escala para el tipo clock_t, pero puede no representar la resolución real del reloj utilizado; es decir, que aunque este numero sea considerablemente alto, la resolucion real del reloj podria ser menor, razon por la cual clock_t podria no tomar todos los valores posibles, sino crecer en pasos mayores a uno.


Ejemplo:
    long int nro;
    cout<<"Ingrese un nro:";
    cin>>nro;
    clock_t t1 = clock();
    for (long int i=0;i<10000;i++)
        for (long int j=0;j<nro;j++)
            ;
    clock_t t2 = clock();
    cout << "Contar diez mil veces hasta " << nro << " toma " << float(t2-t1)/CLOCKS_PER_SEC << " segundos.";
