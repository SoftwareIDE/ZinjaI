Función srand

Inicializa el generador de números pseudo-aleatorios.

Prototipo:
  • void srand ( unsigned int seed );


Se encuentra en: <cstdlib>

    Inicializa el generador de números aleatorios utilizando la semilla seed. Si se utiliza dos veces la misma semilla ambas sucesiones de números aleatorios serán identicas. Si se utiliza una semilla diferente se espera que la sucesión sea completamente diferente.
    Un uso frecuente consiste es inicializar el generador de números aleatorios con la hora actual en segundos (time(NULL)), función que se encuentra en <ctime>).

Ejemplo:
    srand(time(NULL));
    for (int i=0;i++;i<10)
       cout<<rand()<<" ";

  • Cabecera cstdlib: Contiene funciones generales, incluyendo funciones para el manejo de memoria, conversion de tipos, control de proceso, etc.

  • Función time: devuelve la fecha/hora actual en segundos.
