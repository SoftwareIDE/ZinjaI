Función localtime

Convierte una fecha y hora time_t a una fecha y hora tm local.

Prototipo:
  • struct tm * localtime ( const time_t * timer );


Se encuentra en: <ctime>

    Convierte la fecha y hora apuntada por timer (en segundos) a su equivalente para la estructura tm (dividida en componentes). La fecha y hora resultantes son, a diferencia de las retornadas por gmtime, fechas y horas locales. Devuelve un puntero a una estructura de tipo tm. Esta estructura esta alocada estáticamente y es compartida pos las funciones gmtime y localtime, por lo que cualquier llamada a cualquiera de ellas la sobreescribirá.


Ejemplo:
    time_t t1 = time(NULL);
    tm *ptm2 = localtime( &t1 );
    cout << "Fecha actual (local): " << ptm2->tm_mday << "/" << ptm2->tm_mon+1 << "/" << ptm2->tm_year+1900 << endl;
    cout <<"Hora actual (local): " << ptm2->tm_hour << ":" << ptm2->tm_min << ":" << ptm2->tm_sec << endl;


Vea tambien:
  • Cabecera ctime: Contiene funciones y estructuras para el manejo de fechas y horas.
  • Función time: Devuelve la fecha/hora actual en segundos.
  • Función gmtime: Convierte una fecha y hora time_t a una fecha y hora tm UTC.
  • Tipo time_t: Representa una fecha y hora determinada en segundos.
  • Tipo tm: Contiene una fecha y hora determinada separada en sus componentes.
