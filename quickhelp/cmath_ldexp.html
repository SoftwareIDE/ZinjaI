Función frexp

Reconstruye un número a partir de su mantisa y su exponente.

Prototipos:
  • double ldexp ( double m, int e );
  • float ldexp ( float m, int e );
  • long double ldexp ( long double m, int e );


Se encuentra en: <cmath>

    Devuelve un número de punto flotante reconstruido a partir de los valores correspondientes a la mantisa m y el exponente e (mediante la fórmula x=m*2^e).     Esta forma de representar un número está directamente ligada a la forma en que se almacenan en memoria los número de punto flotante (como float o double).


Ejemplo:
    float num=3.5, mantisa;
    int exponente;
    mantisa = frexp(num,&exponente);
    float num_otra_vez = ldexp(mantisa,exponente);
