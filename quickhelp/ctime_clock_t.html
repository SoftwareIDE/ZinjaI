Tipo clock_t

Representa una cantidad de ticks del reloj.

Se encuentra en: <ctime>

    Este tipo de dato esta pensado para representar una cuenta de ticks de reloj y soportar operaciones aritmeticas entre ellas. Se utiliza en funciones de la libreria ctime.


Ejemplo:
    long int nro;
    cout<<"Ingrese un nro:";
    cin>>nro;
    clock_t t1 = clock();
    for (long int i=0;i<10000;i++)
        for (long int j=0;j<nro;j++)
            ;
    clock_t t2 = clock();
    cout << "Contar diez mil veces hasta " << nro << " toma " << float(t2-t1)/CLOCKS_PER_SEC << " segundos.";
