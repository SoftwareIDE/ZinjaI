Función strcspn

Busca en una cadena el primer caracater perteneciente a un conjunto dado.

Prototipos:
  • int strcspn ( const char * str1, const *str2 );
  • int strcspn ( char * str1, const char *str2 );


Se encuentra en: <cstring>

    Recorre la cadena str1 buscando en ella alguno de los caracteres que componen la cadena str2. Si encuentra alguno retorna la posición del mismo, si no encuentra ninguno retorna el largo de la cadena (la posición del caracter de terminación de la cadena).
    La diferencia frente a la función strchr consiste en que, además de poder buscar más de un caracter, strcspn devuelve la posición (un entero con el índice del caracter dentro del arreglo de caracteres), mientras que strchr devuelve un puntero (que puede interpretarse como un cstring que comienza donde se encontró el caracter).


Ejemplo:
   char s[256];
   cout<<"Ingrese una cadena: ";
   cin.getline(s,256);
   int p = strcspn(s,"012345689");
   if (s[p]=='\0')
      cout<<"No hay numeros en esta cadena.\n";
   else
      cout<v"El primer numero esta en la posicion "<<p<<".\n";
