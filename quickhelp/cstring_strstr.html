Función strstr

Busca una cadena dentro de otra.

Prototipos:
  • const char * strstr ( const char * str1, const *str2 );
  • char * strstr ( char * str1, const char *str2 );


Se encuentra en: <cstring>

    Busca la cadena apuntada por str2 dentro de la cadena apuntada por str1 y devuelve un puntero a la primer ocurrencia si la encuentra, o NULL en caso contrario.


Ejemplo:
    // Reemplazar "Hola" por "Chau" en el cstring cadena     char cadena[256]="Y el compilador suicida dijo: \"Hola Mundo!\"\n";
    char *res=strstr(cadena,"Hola");
    if (res!=NULL)
        strncpy(res,"Chau",4);


Vea tambien:
  • Cabecera cstring: Contiene funciones para el manejo de cadenas de caracteres c y arreglos.
  • Función strchr: Busca la primer ocurrencia de un caracter dentro una cadena.
  • Función strrchr: Busca la última ocurrencia de un caracter dentro una cadena.
