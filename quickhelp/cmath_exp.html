Función exp

Calcula la función exponencial en x.

Prototipos:
  • double exp ( double x );
  • float exp ( float x );
  • long double exp ( long double x );


Se encuentra en: <cmath>

    Devuelve el resultado de evaluar la función exponencial en x.


Ejemplo:
    float e = exp(1);
