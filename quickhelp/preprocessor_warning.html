Directiva de preprocesador: #warning

Genera una advertencia en tiempo de compilación.

Utilización:
  • #warning mensaje


    Indica al preprocesador del compilador que debe generar una advertencia con el texto dado por mensaje cuando se intente compilar ese bloque de código.

Ejemplo:
    #warning Este bloque contiene sentencias de código no contemapladas por el estándar de C++


Vea tambien:
  • Directivas de Preprocesador.
  • #error: Genera un error en tiempo de compilación.
  • #line: Permite controlar el nombre de archivo y numero de linea con que se informará un error de compilación.
