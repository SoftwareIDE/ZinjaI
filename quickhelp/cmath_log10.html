Función log10

Calcula el logaritmo en base 10 de x.

Prototipos:
  • double log10 ( double x );
  • float log10 ( float x );
  • long double log10 ( long double x );


Se encuentra en: <cmath>

    Calcula el logaritmo en base 10 de x. El logaritmo natural utiliza como base al número e. El argumento debería ser positivo.
    Matemáticamente, el logaritmo en base 10 de x es un número y tal que 10^y=x. Para encontrar un logaritmo de x en una base arbritaria b, se puede utilizar log10(x)/log10(b).


Ejemplo:
    float dos = log10(100);
