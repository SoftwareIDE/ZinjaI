Función atan2

Calcula el arco tangente de y sobre x.

Prototipos:
  • double atan2 ( double y, double x );
  • long double atan2 ( long double y, long double x );
  • float atan2 ( float y, float x );


Se encuentra en: <cmath>

    Devuelve el arco tangente de x sobre y expresado en radianes en el intervalo [-pi;+pi]. Se utilizan los signos de x e y para determinar a qué cuadrante corresponde.


Ejemplo:
    float x = -1, y = 1;
    float grados = atan2(x/y)*180/M_PI;
