Función modf

Devuelve por separado la parte entera y la parte decimal de un número.

Prototipos:
  • double modf ( double x , double *p_ent );
  • float modf ( float x, float *p_ent );
  • long double modf ( long double x, long double *p_ent );


Se encuentra en: <cmath>

    Separa la parte entera y la parte decimal de un numero x y las devueve por separado. La parte decimal es el valor de retorno de la función, mientras que para la parte entera se utiliza p_ent.


Ejemplo:
    double num = 2.35, p_entera, p_decimal;
    p_decimal = modf(num,&p_entera);
