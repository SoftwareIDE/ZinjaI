Función pow

Calcula una potenciación.

Prototipos:
  • double pow ( double b, double e );
  • float pow ( float b, float y );
  • long double pow ( long double b, long double e );


Se encuentra en: <cmath>

    Calcula la potencia e-esima de b (b^e). b es la base y e el exponente.
    Tambien se puede utilizar para calcular raices, dado que el exponente puede no ser entero (la raiz i-esima de b es b^(1/i) ).
    En algunas circunstancias el compilador puede no llegar a determinar cual de las sobrecargas de esta función utilizar. En caso de ambigüedad, existen las funciones powf y powl para forzar los datos a tipos double y long double respectivamente, aunque estas dos funciones no son estandar.


Ejemplo:
    float x = pow(2.f,10);
    float raiz_cubica = pow(2,1.f/3);
