Directiva de preprocesador: #error

Genera un error en tiempo de compilación.

Utilización:
  • #warning mensaje


    Indica al preprocesador del compilador que debe generar un error con el texto dado por mensaje cuando se intente compilar ese bloque de código.

Ejemplo:
    #if MAX_SIZE<10
    #error El tamaño mínimo MAX_SIZE se ha definido demasiado pequeño.
    #endif


Vea tambien:
  • Directivas de Preprocesador.
  • #warning: Genera una advertencia en tiempo de compilación.
  • #line: Permite controlar el nombre de archivo y numero de linea con que se informará un error de compilación.
