Función strspn

Cuenta cuantos caracteres pertenecientes al conjunto dado hay al comienzo de una cadena. Es decir, busca en una cadena el primer caracater no perteneciente a un conjunto dado.

Prototipos:
  • int strspn ( const char * str1, const *str2 );
  • int strspn ( char * str1, const char *str2 );


Se encuentra en: <cstring>

    Recorre la cadena str1 buscando en ella un caracter que no se encuentre entre los que componen la cadena str2. Si encuentra alguno retorna la posición del mismo, si no encuentra ninguno retorna el largo de la cadena (la posición del caracter de terminación de la cadena).


Ejemplo:
   char s[256];
   cout<<"Ingrese una cadena: ";
   cin.getline(s,256);
   int p = strspn(s,"012345689");
   if (s[p]=='\0')
      cout<<"Solo hay numeros en esta cadena.\n";
   else
      cout<<"Hay un caracter no numerico en la posicion "<<p<<".\n";
