Función realloc

Cambia el tamaño de un bloque de memoria reservado dinámicamente.

Prototipo:
  • void *realloc( void *ptr, size_t num );


Se encuentra en: <cstdlib>

    Cambia el tamaño de un bloque de memoria reservado previamente con malloc, calloc o realloc. Recibe como primer parametro el puntero al comienzo del bloque (ptr), y como segundo el nuevo tamaño en bytes (num). El bloque puede o no permanecer en la ubiación original. En caso de que el nuevo bloque se encuentre en otra posición de memoria, el contenido anterior es copiado a la nueva ubicación, por lo que los datos originales no se modifican. Devuelve un puntero de tipo void* al comienzo del nuevo bloque, o NULL si la memoria no pudo ser reservada. Normalmente el puntero deberá ser convertido (casting) al tipo de dato requerido. Esta memoria deberá ser liberada con free cuando ya no se utilice.

Ejemplo:
    int *arreglo = (int*) malloc(10*sizeof(int));
    for (int i=0;i<10;i++)
       arreglo[i]=i;
    cout<<"Direccion original: "<<arreglo<<endl;
    for (int i=0;i<10;i++)
       cout<<arreglo[i]<<" ";
    cout<<endl;
    arreglo = (int*) realloc(arreglo, 2000*sizeof(int));
    cout<<"Nueva direccion: "<<arreglo<<endl;
    for (int i=0;i<20;i++)
       cout<<arreglo[i]<<" ";
    cout<<"..."<<endl;
    free (arreglo);


Vea tambien:
  • Cabecera cstdlib: Contiene funciones generales, incluyendo funciones para el manejo de memoria, conversion de tipos, control de proceso, etc.
  • Función malloc: Reserva dinámicamente un bloque de memoria.
  • Función calloc: Reserva e inicializa dinámicamente memoria para un arreglo.
  • Función free: Libera un bloque de memoria reservado dinámicamente.
