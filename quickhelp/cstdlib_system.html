Función system

Ejecuta un comando o programa externo.

Prototipo:
  • int system ( const char* cmd );


Se encuentra en: <cstdlib>

    Invoca al intérprete de comandos del sistema operativo para ejecutar el comando especificado en cmd (una cadena de caracteres c). Devuelve el codigo de salida del programa/comando ejecutado (normalmente 0 indica exito, otro valor indica error), o -1 si no pudo ejecutarlo. Notar que esta ejecución es síncrona, es decir, que el programa esperará a que termine de ejecutarse el comando.

Ejemplo:
    string archivo;
    system("dir");
    cout<<"Ingrese el nombre de un archivo: ";
    getline(cin,archivo);
    int ret;
    if (archivo=="")
       ret = system("notepad.exe");
    else
       ret = system((string("notepad.exe ")+archivo).c_str());
    switch (ret) {
    case 0:
       cout<<"Programa lanzado correctamente"<<endl;
       break;
    case -1:
       cout<<"Error al lanzar el programa"<<endl;
       break;
    default:
       cout<<"El programa se ejecutó y devolvió: "<<ret<<endl;
    }
