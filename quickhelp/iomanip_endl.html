Manipulador endl

Inserta el caracter de fin de linea.

Prototipos:
  • ostream& endl ( ostream &f )


Se encuentra en: <iostream>

    Este manipulador de flujos inserta el caracter de fin de linea ('\n') en el flujo f. Si dicho flujo utiliza un buffer, además sincroniza el buffer con la salida del mismo (al igual que flush).


Ejemplo:
    cout<<"1) Esta es un linea."<<endl<<"2) Esta es otra linea"<<endl
    cout<<"3) Esta es una ";
    cout<<"sola linea."<<endl;
    cout<<"4) Esta es la cuarta linea."<<endl;


Vea tambien:
  • Cabecera iomanip: Contiene manipuladores de flujo para dar formato a flujos de salida.
  • Cabecera iostream: Contiene clases y objetos para el manejo de flujos de entrada y salida.
  • Clase ostream: Modela un flujo de salida genéricos.
  • Clase ofstream: Modela un flujo de salida hacia un archivo.
  • Objeto cout: Representa el flujo de salida estándar.
