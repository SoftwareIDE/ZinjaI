Función main

Es la función de comienzo de todo programa C++.

Prototipos:
  • int main (void);
  • int main (int argc, char *argv[]);


    Todo programa C++ debe implementar directa o indirectamente la función main (en algunos casos se define a travez de macros o funciones de librerías, o utilizando la función WinMain en su lugar cuando se trata de una aplicación Windows). Es la función que se ejecuta al comenzar el programa, por lo que el enlzador no puede terminar de construir un programa si no existe en este una función main.
    El valor de retorno debe ser un entero (aunque algunos compiladores aceptan void, este no esta avalado por el estándar). Este entero es el código de error que se le devuelve al sistema operativo al finalizar el programa (excepto cuando se utilice la función exit o alguna equivalente). Un valor de retorno igual a cero suele utilizarse para indicar que el programa finalizó correctamente, mientras que un valor distinto de cero suele indicar que ocurrió algún error durante la ejecución del mismo. Los rangos de valores admitidos varían de acuerdo al sistema operativo.
    El segundo prototipo presentado se utliza para recibir parámetros (por ejemplo, los parámetros que siguen al nombre del programa en una llamada desde la línea de comandos). El entero argc indica la cantidad de parámetros recibidos, mientras que las cadenas argv[i] contienen los mismos. Cabe aclarar que argc siempre es mayor o igual a uno, ya que argv[0] siempre contiene la llamada al ejecutable, es decir, su nombre (incluyendo la ruta de ser necesario), por lo que los parámetros propiamente dichos comienzan a partir de argv[1] y son en total argc-1.

Ejemplo:
    int main(int argc, char *argv[]) {
        if (argc==1) {
            cerr<<"Error: se necesita al menos un argumento!\n";
            return 1;
        }
        cout<<"Ejecutable: "<<argv[0]<<endl;
        for (int i=1;i<argc;i++)
            cout<<"Argumento "<<i<<": "<<argv[i]<<endl;
        return 0;
    }
