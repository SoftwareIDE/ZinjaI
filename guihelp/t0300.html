    En este primer tutorial relacionado a la depuración de programas, se van a ilustrar los mecanismos básicos para ejecutar e interrumpir el programa, así como también la inspección de variables. Para ello, se utilizará el siguiente programa ejemplo:

#include <iostream>
using namespace std;
 
int main(int argc, char *argv[]) {
   unitbuf(cout);
   int cant,n;
   cout<<"Cantidad de datos: ";
   cin>>cant;
   double sum=0;
   for (int i=0;i<cant;i++) {
      cout<<"Ingrese el dato "<<i<<": ";
      cin>>n;
      sum+=n;
   }
   double prom = sum/cant;
   cout<<"Promedio: "<<prom;
   return 0;
}
