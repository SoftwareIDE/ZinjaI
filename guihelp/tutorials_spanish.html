Tutoriales

    Los tutoriales pretenden desarrollar un ejemplo práctico guiado paso a paso para mostrar el funcionamiento básico del enterno y algunas de sus características especiales más importantes.

    Los tutoriales disponibles son:

  • Descubriendo Zinjai: introduce el uso del entorno para resolver problemas simples rapidamente.
    • Mi primer programa
    • Características especiales

  • Introducción al uso de Proyectos: introduce el manejo de proyectos para desarrollos de mayor complejidad.
    • Mi primer proyecto
    • Utilizando bibliotecas externas
    • Interfases de usuario con wxFormBuilder
    • Como construir tu propia biblioteca

  • Nociones Básicas de Depuración: presenta las principales funcionalidades de depuración integradas en el entorno.
    • Control básico e inspección de variables
    • Funciones y ámbitos
    • Visualización de estructuras de datos



    Nota: Estos tutoriales no están pensados para aprender a programar, ni para aprender C++, sino que su objetivo es presentar el uso del IDE. Se asume que el lector posee conocimientos mínimos de programación en general y C/C++ en particular.
