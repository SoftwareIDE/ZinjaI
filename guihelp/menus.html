Menúes de la ventana principal

  • Menú Archivo: Contiene acciones relacionadas a la apertura, el guardado y el cierre de archivos (tanto fuentes como proyectos), y acceso a las preferencias generales tanto del entorno como de un proyecto.
  • Menú Editar: Contiene acciones de edición que operan sobre el código fuente (tanto acciones básicas como copiar y pegar, como acciones específicas para la programación con C/C++ como comentar o indentar un código), y comandos para realizar diferentes tipos de búsquedas sobre un fuente o sobre los archivos de un proyecto.
  • Menú Ver: Permite modificar la apariencia del entorno (haciendo visibles u ocultando los diferentes paneles y barras de herramientas que lo componen) y la vista de un código (plegado, coloreado, ajuste de linea, etc).
  • Menú Ejecucion: Contiene los comandos básicos para iniciar y configurar la compilación y ejecución de un proyecto o programa.
  • Menú Depuracion: Contiene todos los comandos relacionados a la configuración y el control de una sesión de depuración.
  • Menú Herramientas: Agrupa herramientas adicionales varias, tanto las provistas por el IDE como también las herramientas de terceros para las cuales ZinjaI ofrece facilidades de integración y uso en sus propios proyectos.
  • Menú Ayuda: Permite acceder a las distintas páginas de ayuda que ofrece el entorno (sobre el uso del mismo, sobre C/C++, sobre el código que se está editando, etc), y permite además consultar la información de licencias y versiones.
