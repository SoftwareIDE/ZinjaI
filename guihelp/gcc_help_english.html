Compiler's documentation

    ZinjaI, by default use GCC compiler (http://gcc.gnu.org/). You can access the full gcc manual at http://gcc.gnu.org/onlinedocs/, but as a ZinjaI user you might be particulary interested in its command line arguments. Help for latest gcc's command line arguments can be found on http://gcc.gnu.org/onlinedocs/gcc-4.9.1/gcc/Invoking-GCC.html.
