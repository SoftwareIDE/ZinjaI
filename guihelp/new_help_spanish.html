Si va realizar un programa pequeño, utilizando un solo archivo fuente (.cpp) seleccione "Utilizar Plantilla". Si va a realizar un programa de tamaño considerable, o desea poder configurar con gran detalle las opciones de compilación y ejecución seleccione "Proyecto".
  • Archivo en Blanco: Crea un nuevo archivo completamente en blanco.
  • Utilizar Plantilla: Crea un nuevo archivo con un esqueleto basico de programa (crea la funcion main e incluye cabeceras acordes al tipo de programa seleccionado, y si es necesario configura opciones basicas de compilación y ejecución.
  • Proyecto Nuevo: A diferencia de un programa simple, el proyecto puede constar de mas de un archivo fuente y guarda tambien perfiles de configuración detallados (parametros para el compilador, bibliotecas a enlazar, integración de herramientas externas, etc.).
  • Proyecto Existente: Use esta opción para crear un proyecto ZinjaI utilizando un conjunto de fuentes ya existentes (para migrar desde otros IDEs o herramientas como make).
Para mayor detalle, utilice el botón de ayuda ubicado en la esquina inferior izquierda para ir directamente a la pagina relacionada del sistema de ayuda.
