Documentación con Doxygen

    Doxygen es una herramienta para la generación automatizada de documentación. La documentación se genera a partir del análisis del código fuente y/o sus comentarios (existen formatos de comentarios especiales para doxygen con sus propias palabras claves). Para descargar el programa u obtener mayor información visite www.doxygen.org.

    La configuración acerca de como generar la documentación se toma a partir de un archivo de texto generalmente denominado "Doxyfile". ZinjaI ofrece un cuadro de diálogo simplificado para generar dicho archivo aprovechando las opciones conocidas tomadas desde la configuración de un proyecto, y accesos rápidos para regenerar o abrir la documentación sin salir del entorno. Sin embargo, el instalador de ZinjaI no incluye Doxygen, por lo que antes de utilizar estas características deberá descargarlo e instalarlo, y en caso de instalarlo en una ruto no estándar, ingresar la ubicación de su ejecutable en la pestaña Rutas 2 del cuadro de diálogo de Preferencias.
