Ayudas en la Escritura

    ZinjaI presenta un amplio conjunto de funcionalidades especialmente destinadas a facilitar la escritura y edición de un código fuente:

  • Edicion Inteligente: consiste en la indentación automática del código a medida que se ca escribiendo
  • Plegado de Código: permite ocultar o mostrar bloques lógicos de código como clases, funciones, bucles, etc
  • Coloreado de Sintaxis: el código se visualiza utilizando diferentes colores para cada elemento del mismo para facilitar su lectura
  • Autocompletado: en determinados casos se muestra automáticamente un menú con los nombres de las posibles funciones, metodos o variables para que el usuario pueda ingresarlos más rápidamente
  • Ayuda Para Llamadas: cuando se abre un paréntesis para ingresar los parámetros de una función ZinajI puede mostrar un globo de ayuda recordando los numeros y tipos de parametros de la función y sus posibles sobrecargas.
  • Plantillas de Auto-Código: son fragmentos de código definidos de forma similar a una macro de preprocesador, que ZinjaI los reemplaza automáticamente cuando se invocan al escribir un programa.
  • Otros complementos: existen además otros comandos de edición (por ejemplo para comentar/descomentar un bloque, o duplicar líneas, insertar #includes, etc) que también facilitan el proceso de edición.
  • Atajos de teclado: tabla resúmen de atajos de teclado.
