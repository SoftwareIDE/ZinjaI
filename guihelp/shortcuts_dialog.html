Personalizar Atajos de teclado

Este cuadro de diálogo permite configurar los atajos de teclado para las distintas acciones disponibles en los menúes de ZinjaI y un conjunto de acciones adicionales. Se accede al mismo mediante el botón "Personalizar atajos de teclado..." de la pestaña General del cuadro de diálogo Preferencias. Esta ventana presenta la lista de acciones posibles, cada una acompañada de un caudro de texto donde puede introducir el atajo a asignar, y un botón (...) que permite capturar el atajo (es decir, abre una ventana donde pulsar la combinación de teclas a asignar). En la parte superior de la ventana puede introducir parte del nombre de un comando para hacer una búsqueda, o un atajo de teclado para ver a qué comando está asignado. Para introducir un atajo como texto pude ingreser el nombre de una o más teclas, utilizando el signo + para combinar más de una (ej: Ctrl+Alt+Z). Si quiere que una acción no tenga un atajo asociado puede dejar el cuadro de texto correspondiente a la misma en blanco.
