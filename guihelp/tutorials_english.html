Tutoriales

    This tutorial are intended to develop guided practical examples step by step to show basis about programming with ZinjaI and some of its most usefull special features.

    Available tutorials are:

  • Discovering Zinjai: shows how to use this IDE to easily sole simple programming problems.
    • My first program
    • Special features

  • Introduction to Projects: presents the use of projects to develop more complex programs.
    • My first project
    • Using external libraries
    • User interfaces with wxFormBuilder
    • How to build your own library

  • Debugging basis: explaing how to take advantage of main debugging features.
    • Basic control and inspection
    • Functions and scopes
    • Data structures visualization



    Note: This tutorials are not meant to teach neither C++ nor programming. This tutorials are made to expose main ZinjaI's features as an IDE, and to understand them user should have some basic programming and C++ knowledge.
