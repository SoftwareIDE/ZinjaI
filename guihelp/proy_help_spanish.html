Este diálogo permite gestionar perfiles de configuración de proyecto (puede definir uno o más perfiles). Para seleccionar o agregar estos perfiles utilice los controles de la parte superior de la ventana. Cada perfil define los siguientes aspectos:
  • General: define opciones generales (ubicacion de temporales y ejecutables) y de ejecución (argumentos y directorio de trabajo) del proyecto
  • Compilacion: define los parametros que se utilizan para la compilación (directorios de cabeceras, macros definidas, optimización y debugging, etc.)
  • Enlazado: define los parametros que se utilizan para el enlazado (bibliotecas, ubicacion de las mismas, si utiliza la consola, etc.)
  • Secuencia: permite modificar el proceso de compilación para agregar pasos especiales (archivos de recurso, parsers especificos de librerias, etc).
  • Bibliotecas: permite construir bibliotecas a partir de un subconjunto de fuentes del proyecto.
Para mayor detalle, utilice el botón de ayuda ubicado en la esquina inferior izquierda para ir directamente a la pagina relacionada del sistema de ayuda.
