Este diálogo permite gestionar perfiles de configuración de proyecto (puede definir uno o más perfiles). Para seleccionar o agregar estos perfiles utilice los controles de la parte superior de la ventana. Cada uno define los siguientes aspectos:
  • General: define opciones generales (ubicacion de temporales y ejecutables) y de ejecución (argumentos) del proyecto
  • Compilacion: define los parametros que se utilizan para la compilación (directorios de cabeceras, macros definidas, optimización y debugging, etc.)
  • Enlazado: define los parametros que se utilizan para el enlazado (librerias, ubicacion de las librerias, si utiliza la consola, etc.)
  • Secuencia: permite modificar el proceso de compilación para agregar pasos especiales (archivos de recurso, parsers especificos de librerias, etc).
Para mayor detalle, utilice el botón de ayuda ubicado en la esquina inferior izquierda para ir directamente a la pagina relacionada del sistema de ayuda.
