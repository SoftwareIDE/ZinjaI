Diálogo Generar Biblioteca




    Este diálogo permite configurar la generación de una biblioteca (para obtener más informacion acerca del proceso completo y la generación bibliotecas en proyectos de ZinjaI consulte Creación de Bibliotecas). Las opciones asociadas a una biblioteca a generar son:
    1. Nombre: el nombre de la biblioteca define el nombre del archivo de la misma. Este se forma anteponiendo el prefijo lib y añadiendo la extensión adecuada al final (varía segun sistema operativo y tipo de enlazado).
    2. Directorio de salida: indica en que directorio se colocará la biblioteca (el archivo binario).
    3. Opciones de enlazado: estos argumentos se utilizan al enlazar los objetos de los fuentes para construir la biblioteca (los parámetros de la pestaña Enlazado del cuadro de Opciones de Compilación y Ejecución de Proyecto sólo se aplican al ejecutable, no a las bibliotecas). Vea Ayuda del compilador utilizado.
    4. Tipo de biblioteca: define si la bibioteca será enlazada dinámica o estáticamente.
    5. Fuentes a Incluir/Excluir: estas listas indican qué archivos fuentes deben considerarse (compilarse y enlazarse) al construir la biblioteca. Los botones en medio de las listas permiten pasar fuentes de una a otra.
    6. Biblioteca por defecto: esta casilla sólo puede estar activa para una biblioteca. Cuando se añada un archivo fuente nuevo al proyecto, si alguna biblioteca tiene esta opción activada, el nuevo fuente pasa a formar parte de la lista Fuentes a Incluir para esa biblioteca.
