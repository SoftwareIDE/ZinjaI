Menú Ayuda: Acerca de...

    Muestra un cuadro de diálogo con el logo del software e información acerca de su versión y sitio web.
     Menú Ayuda: Tutoriales...

    Abre la ventana de ayuda y muestra la página que contiene el índice de tutoriales sobre el uso del entorno.
     Menú Ayuda: Ayuda sobre ZinjaI...

    Abre esta ventana de ayuda y muestra el índice de la misma. Aquí se puede encontrar ayuda acerca del uso y configuración del entorno, así como documentación acerca de la licencia, versión, sitio web, etc.
     Menú Ayuda: Referencia C/C++...

    Permite obtener ayuda rápida acerca de una librería, función, clase, estructura, palabra reservada, etc del lenguaje. Esta ayuda no se incluye en el instalador de ZinjaI, sino que debe descargarse por separado e instalarse como complemento. Al intentar ingresar a la misma por primera vez ZinjaI abrirá un navegador en el sitio de descarga y mostrará un cuadro de diálogo para la instalación del complemento.
    Esta primer referencia solo muestra contenido C/C++ estándar. ZinjaI posee además un panel de ayuda rápida donde puede acceder fácilmente descripciones sobre los símbolos (clases, métodos, funciones, etc.) que defina en su proyecto, autogenerada por ZinjaI a partir del análasis de los fuentes, y complementada con la documentación generada por Doxygen si está disponible y actualizada. Para acceder a la misma debe presionar Shift+F1. Para que esta segunda ayuda acerca del propio código esté disponible se debe mantener actualizado el Arbol de simbolos (lo cual se hace automáticamente al guardar/ejecutar, o manualmente con la tecla F2).
     Menú Ayuda: Mostrar Sugerencias de uso...

    Muestra el cuadro de diálogo de sugerencias. El mismo contiene breves sugerencias acerca del uso de ZinjaI y sus prinicipales características, así como algunos trucos y consejos para aprovechar mejor sus opciones y herramientas. Este cuadro es el que se muestra cuando se carga el entorno si la opción Mostrar sugerencias al inicio de la pestaña General del cuadro de diálogo Preferencias (Archivo->Preferencias... ó Ctrl+P) está seleccionada.
     Menú Ayuda: Enviar sugerencia o reportar error...

    Abre la sección de foros del sitio oficial de Zinjai en navegador del sistema, para poder enviar un mensaje al desarrollador de ZinjaI y/o la comunidad de usuarios. El sitio dispone de diferentes foros según el objetivo del mensage, existiendo foros específicos para consultas sobre el uso del IDE, para reporte de errores encontrados en el mismo, y para enviar sugerencias y comentarios libres.
     Menú Ayuda: Buscar Actualizaciones...

    Intenta conectarse a travéz de internet para determinar si hay disponible en el sitio web de ZinjaI una versión más reciente que la que se está utilizando. En caso afirmativo permite observar el registro de cambios. Este método no funciona si el sistema utiliza un proxy para navegar.
