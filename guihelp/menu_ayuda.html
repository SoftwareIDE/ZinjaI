 Menú Ayuda: Ayuda sobre C++...

    Permite obtener ayuda rápida acerca de una librería, función, clase, estructura, palabra reservada, etc del lenguaje. Cuando se selecciona esta opción o se presiona Shift+F1, se busca ayuda sobre la palabra en la que está actualmente posicionado el cursor en el código. Primero se busca dentro de la ayuda del lenguaje estándar. Si se encontró la palabra buscada se muestra la ayuda correspondiente. Si no se encuentra, se busca luego si la palabra seleccionada es una macro, función, clase, método o atributo definido en el programa. De ser así se muestra un breve descripción del mismo que incluye su prototipo, su jerarquia, su visibilidad y/o sus métodos y atributos según correspondan, junto con enlaces a las lineas de código en donde se declara y/o define. Para que esta ayuda acerca del propio código esté disponible se debe tener actualizado el árbol de símbolos, ya que dicha ayuda se genera durante este proceso de actualización.

 Menú Ayuda: Mostrar Sugerencias...

    Muestra el cuadro de diálogo de sugerencias. El mismo contiene breves sugerencias acerca del uso de ZinjaI y sus prinicipales características, así como algunos trucos y consejos para aprovechar mejor sus opciones y herramientas. Este cuadro es el que se muestra cuando se carga el entorno si la opción Mostrar sugerencias al inicio de la pestaña General del cuadro de diálogo Preferencias (Archivo->Preferencias... ó Ctrl+P) está seleccionada.
Menú Ayuda: Buscar Actualizaciones...

    Intenta conectarse a travéz de internet para determinar si hay disponible en el sitio web de ZinjaI una versión más reciente que la que se está utilizando. En caso afirmativo permite observar el registro de cambios. Este método no funciona si el sistema utiliza un proxy para navegar.
