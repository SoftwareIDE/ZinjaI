Atajos de teclado

La siguiente tabla muestra un resúmen de atajos de teclado para distintas operaciones de edición y otras acciones en ZinjaI (la lista no incluye todos los atajos disponibles).

Importante: la lista muestra los atajos en su configuración por defecto; puede modificarlos desde el cuadro de diálogo Atajos de Teclado.


Manejo de Archivos/Pestañas de código
Ctrl+NNuevo programa simple o nueva archivo en proyecto
Ctrl+Shift+NNuevo proyecto
Ctrl+OAbrir archivo/proyecto
Ctrl+Tab o Ctrl+AvPagCambiar a la siguiente pestaña de código
Ctrl+Shift+Tab o Ctrl+RePagCambiar a la pestaña de código anterior
Alt+F11Activar/Desactivar ajuste de lineas
Ctrl+SGuardar archivo actual
Ctrl+Shift+SGuardar archivo actual con otro nombre
Ctrl+Shift+Alt+SGuardar todos los archivos abiertos
Ctrl+Alt+SGuardar proyecto
Ctrl+WCerrar pestaña actual
Ctrl+Alt+Shift+WCerrar todas las pestañas
Ctrl+Shift+WCerrar proyecto actual
Ctrl+Shift+RRecargar archivo actual
Selección y desplazamiento

Alt+IzquierdaRetroceder en el historial de posiciones del cursor de texto

Crtl+GIr a un número de linea en particular
Crtl+Shift+GIr a la definición de una función/método/clase
Crtl+Shift+FIr a un archivo abierto o del proyecto
Crtl+EnterAbrir el archivo seleccionado o sobre el cual se encuentra el cursor
F12Abrir el .cpp homónimo estando en un .h, o el .h estando en un .cpp
Ctrl+>Ir al siguiente error de compilación

Alt+ArribaPlegar un bloque de código

Edición por lineas y bloques
Ctrl+LDuplicar la línea actual o las lineas seleccionadas
