This windows lets you define settings related to the aspect and behavoiur of ZinjaI. This settings are sorted by category (icons on the left). Categories are:
  • General: general behaviour and appearence system settings.
  • Simple Program: options related to creation and execution of simple programs (it means, without projcets).
  • Style: code displaying options (colours, sizes, tabulations, ...).
  • Writing: let's you enable and disable several assistances that ZinjaI offers you for writing code (calltips, autocompletion, indentaton, ...).
  • Icon theme: wich set of icons and images to use.
  • Toolbars: let's you choose wich toolbars to display and wich buttons to place on them.
  • Paths 1: folders where ZinjaI looks for its files (help, templates, projects, ...) or write temporal ones.
  • Paths 2: comandos que ZinjaI utiliza para llamar a otros programas que se comunican con la IDE, como el compilador, el depurador, la consola, el visor de imagenes, etc.ZinjaI.
  • Debugging: interface's aspect and behaviour while debugging a program, and some debugger features.

For a more detailed help click the help button (lower right corner) to go this topic in ZinjaI's help system.
