    Welcome to ZinjaI's help system. Help pages are organized in the following groups:

  • Tutorials: This section contains some basic tutorials that will guide you step by step to know main ZinjaI's features and learn the basic stuff to start programming with this IDE.
  • Interface: Contains information about ZinjaI's interface (windows, dialogs, menus) organized by window.
  • Special Features: This topics describe main ZinjaI's features that make it different from other editors. Most of them are specifically designed for programming tasks and for teaching programming.
  • Debugging: This section covers topics about debugging in general and about ZinjaI's debugging features.
  • About ZinjaI: Contains license and other general information about ZinjaI and some other third-part software that are included in ZinjaI. It also contain's a detailed change-log that describe improvements, bug-fixes and new features for every release.
